Welcome to HelloWorld-API!

This is a simple REST API containing "Hello world" print statements in several computer languages.

Go to API Go to GitHub repo

Json example:


    "Python": [
        {
            "Code":"print(\"Hello World!\")"
        }
    ],

Get data using Python


    #!/usr/bin/python3
    import requests
            
    req = requests.get("http://localhost:5500/APIs/HelloWorld.json")
    dct = req.json()
    
    pst = dct["Python"]
    pst2 = dct["Java"][0]["Code"]
    print(pst)
    print(pst2)

    OUTPUT:
    [{'Code': 'print("Hello World!")'}]
    System.out.println("Hello World!")
